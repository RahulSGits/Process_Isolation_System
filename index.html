Deadlock Recovery Techniques

Methods for system recovery from detected deadlocks

Process Termination

Abort one or more processes involved in the cycle. Safest approach but may lose work.

Complexity: O(n) where n is number of processes in cycle

Resource Preemption

Preempt resources from one process and give to another. Challenging with non-preemptible resources.

Complexity: O(m) where m is number of resources

Safe State Rollback

Rollback system to previous safe state. Requires checkpointing and logging system state.

Complexity: O(log n) with binary search on checkpoints

Optimal Resource Allocation

Reorganize resource allocation to break the cycle. Requires minimal process interruption.
